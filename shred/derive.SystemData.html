#[derive(SystemData)]

A reexport of the #[derive(SystemData] macro provided by shred-derive. This requires that the shred-derive feature is enabled. Used to #[derive] the trait SystemData.

You need to have the following items included in the current scope:

  • SystemData
  • World
  • ResourceId

This macro can either be used directly via shred-derive, or by enabling the shred-derive feature for another crate (e.g. shred or specs, which both reexport the macro).
